<Subheading>Basic link</Subheading>
<Subtitle>
  Link component is an extension of HTML anchor element. It supports all the
  props of its HTML counterpart.
</Subtitle>

<Preview>
  <Story name="Basic link">
    <div style={{ padding: "10px" }}>
      <Link href="#">Take me places!</Link>
    </div>
  </Story>
</Preview>

<Subheading>External link</Subheading>
<Subtitle>
  It is possible to indicate that a link is external by providing external prop.
</Subtitle>

<Preview>
  <Story name="Basic usage">
    <div style={{ padding: "10px" }}>
      <Link href="#" external>
        Take me places!
      </Link>
    </div>
  </Story>
</Preview>

<Subheading>Link with an icon</Subheading>
<Subtitle>
  It is possible to add an icon before link text by providing icon prop.
</Subtitle>

<Preview>
  <Story name="Link with an icon">
    <div style={{ padding: "10px" }}>
      <Link href="#" icon={<FavouriteIcon />}>
        Take me places!
      </Link>
    </div>
  </Story>
</Preview>

<Subtitle>The link will inherit colours from its parent component.</Subtitle>

<Preview>
  <Story name="Colour inheritance">

<Subheading>Polymorphism</Subheading>
<Subtitle>
  The base element in a link component is polymorphic, meaning that it is
  possible to render any other HTML element as a base element in a link
  component. This is useful when rendering link elements from third-party
  routing libraries.
  <br />
  <br />
  Note: it is important not to break semantics.
</Subtitle>

<Preview>
  <Story name="Polymorphism">
    <div style={{ padding: "10px" }}>
      <Link href="#" as="span">
        Take me places!
      </Link>
    </div>
  </Story>
</Preview>
